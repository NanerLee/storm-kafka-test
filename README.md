# Storm-Kafka-Test

本代码是使用 storm-kafka-client 消费和生产 kafka 数据的示例, 包括非安全模式和使用 kerberos 认证的安全模式

## 环境

- jdk-1.7 及以上
- storm-1.0.6
- kafka-0.10.0.0

## 详解

具体详解参考我的这边文章 [Storm 访问使用 Kerberos 认证的 Kafka 集群](https://blog.nanerlee.me/2018/09/29/storm-kafka-kerberos/)
